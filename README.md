# ✨ gorrent ✨
gorrent使用Go语言进行开发，是一个实现了基础BitTorrent协议的客户端，能够解析`.torrent`文件并下载对应的资源文件。

## 🚀 快速开始
1. 克隆项目
```sh
git clone https://github.com/JmZeroQAQ/gorrent.git
```

2. 切换到cmd目录
```sh
cd gorrent/cmd
```

3. 编译项目
```sh
go build -o gorrent main.go
```

4. 运行
```sh
./gorrent ../test/debian-iso.torrent
```
